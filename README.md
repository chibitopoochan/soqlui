# soqlui
SOQLをローカルで実行するクライアントツール

[![MIT licensed][shield-license]]

## Description
ブラウザを起動せずにローカルで実行できるSOQLのクライアントツール。

## Usage
### Install
インストーラー「SOQLUI-x.x.exe」を実行してください。

### Run
SOQLUIを実行します。
または拡張子「soql」を開きます。

### Check
```sh
firefox http://localhost:8080/my-project &
```
とか
